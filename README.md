This is my *mostly complete* starting point for a Love2D game that uses game states and a CSV file for saving game data.  

The fully playable example game is intentionally made without libraries, images, or sound - it's just made with [LÖVE](https://www.love2d.org/)!

I made this template, as a complete game, because I would like to make an arcade style to play on [Anbernic RG 40XXV](https://retrocatalog.com/retro-handhelds/rg-40xxv) and making this gets me back into the swing of things.  

There are some Love2D games on [PortMaster](https://portmaster.games/), so I assume this one will run fine too. Hopefully it does, because I much prefer the simplicity of programming SDL2 in Lua than in C/C++! I considered using [GB Studio](https://www.gbstudio.dev/) to make a GameBoy Color game, but I have a lot more experience with Lua and enjoy my existing workflow and tools.  

I made this for LÖVE 11.5, as provided in [Debian 13 "Trixie"](https://www.debian.org/intro/why_debian).
